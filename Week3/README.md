# Week 3: The Insight (NLP) & Mid-Sem Evaluation

**Focus:** Sentiment analysis with TextBlob and full pipeline integration.

This week introduced lexicon-based sentiment analysis — polarity (negative to positive, -1.0 to +1.0) and subjectivity (fact to opinion, 0.0 to 1.0). The mid-semester deliverable is a single notebook demonstrating the complete linear pipeline: load an image, resize it with Pillow, extract text with EasyOCR, analyze sentiment with TextBlob, and display the image with its extracted text and sentiment scores.
