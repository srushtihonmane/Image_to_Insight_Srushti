# Week 1: The Digital Canvas

**Focus:** Python setup and image manipulation with Pillow.

This week covered how computers represent images as matrices of numbers and why preprocessing matters for OCR. The assignment notebook loads an image, prints its original dimensions, converts it to grayscale, resizes it to 50% of the original scale, and saves the result.

## Files

- `Assignment1.ipynb` — setup check (Assignment 1.1) and the `process_meme()` preprocessing pipeline (Assignment 1.2)
- `sample.jpg` — original input image
- `processed_week1.jpg` — grayscaled, half-scale output produced by the pipeline
